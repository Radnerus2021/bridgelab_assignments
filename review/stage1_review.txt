Algorithm for copyfiles program.

*Created a multiple files in a respective folder.

1. use for loop for fetching files recursively.
2. created a variable "folderName" and added value to it.
3. Create a variable "Subfolder" and added value to it.
4. Using if condition, checked and deleted the existing similar directory.
5. Using mkdir created a folder using each variable.
6. Using cp command copied files into subfolder
7. Using mv command moved subfolder into folderName
8. Using echo printed the sturcture required.
